Project Introduction

This project aims to leverage the power of deep learning, specifically Convolutional Neural Networks and Long Short-Term Memory (CNN-LSTM) model, to predict the closing stock prices for a one-day forward period based on time series data. Additionally, the project integrates technical analysis-based recommendations to guide investors while making buy, hold, or sell decisions.



Objective

Key Objectives:

1. Develop a CNN-LSTM model to predict stock prices for the next trading day based on historical price data.

2. Incorporate technical analysis indicators to generate buy, hold, or sell recommendations.



Methodology

1. Data Collection: Gather historical stock price data using yfinance(python wrapper for Yahoo Finance).

2. Data Preprocessing: Clean the collected data, handle missing values, and decompose the data to check seasonality, trend, and residual factors.

3. Feature Engineering: Select and engineer key features, including technical analysis indicators such as simple moving averages, exponential moving average, relative strength index (RSI), MACD, BIAS, Stochastic K, Stochastic D. And time series component including trend, seasonality, and residual variables.

4. Model Development: Design, fine-tune model using the preprocessed data to predict the next day's stock prices.

5. Model Evaluation: Assess the model's performance by comparing the predicted prices against the actual stock prices. Using metrics such as Root Mean Squared Error (RMSE) and r-squared.

6. Recommendation: This section use technical analysis indicators to generate buy, hold, or sell recommendations based on predefined rules and thresholds.

Expected Outcome

1. A trained CNN-LSTM model capable of predicting stock prices with reasonable accuracy.

1. A technical analysis-driven recommendation system that suggests buy, hold, or sell actions based on the predicted prices and technical indicators.
